# Recursos del perfil

Coloca aquí los archivos gráficos de la escuela y referéncialos desde
`../config/branding.ts`. Nombres por rol esperados (puedes ajustar las
extensiones según tus archivos):

| Archivo | Uso |
|---|---|
| `logo-universidad.png` | Logo de la universidad (navbar + footer) |
| `logo-escuela.png` | Logo de la escuela (navbar) |
| `logo-bolsa-trabajo.png` | Ícono de bolsa de trabajo (hero) |
| `libro-reclamaciones.svg` | Botón Libro de Reclamaciones (hero) |
| `hero/hero-1.webp` | Imagen de fondo del hero (1) |
| `hero/hero-2.webp` | Imagen de fondo del hero (2) |
| `hero/hero-3.webp` | Imagen de fondo del hero (3) |

Las imágenes del hero se recomiendan en `.webp` y orientación horizontal.
Documentos descargables (p. ej. `Plan_de_Estudios.pdf`) van en `public/`.
